> Os **Segredos** são strings de 11 caracteres usadas para validar se as requisições ao servidor provêm de um cliente Geometry Dash. Em vez de gerar um novo segredo para cada requisição enviada ao servidor, RobTop optou por codificar estes segredos e obscurecê-los um pouco, para evitar que fossem descobertos.

- **Abaixo está uma tabela com todos os segredos conhecidos**

### Tabela de Segredos

|     Segredo    |  Tipo      |
|:--------------|:-----------|
| `Wmfd2893gb7` | Comum      |
| `Wmfv3899gc9` | Conta/MP   |
| `Wmfv2898gc9` | Nível      |
| `Wmfp3879gc3` | Mod        |
| `Wmfx2878gb9` | Admin      |

### Tipos de Segredos
<u>**A partir da versão 2.206 do Geometry Dash, existem 4 segredos *publicamente* conhecidos**</u>

- Segredo Comum
- Segredo de Conta
- Segredo de Nível
- Segredo de Mod

**<h3><u>Segredo de Conta</u></h3>**  
> A partir da versão 2.206 do Geometry Dash, existem 5 endpoints que utilizam o `Segredo de Conta`. O Segredo de Conta foi criado para endpoints que lidam com <u>funcionalidades importantes da conta</u>. Isso inclui: salvar dados, restaurar itens ou ações associadas à conta do jogador.
