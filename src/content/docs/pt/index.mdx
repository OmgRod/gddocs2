> Se não encontrares o que procuras aqui, junta-te ao nosso [Servidor Discord](https://discord.gg/gd-programming-646101505417674758) e podemos ajudar-te!

## Contribuições

> Se quiseres contribuir para este projeto, submete um [Pull Request](https://github.com/OmgRod/gddocs2) no nosso repositório do GitHub

## Projetos da Comunidade

> Ao longo dos anos, muitos membros da comunidade desenvolveram as suas próprias ferramentas e software que interagem com o Geometry Dash e os seus servidores

### Ferramentas e Software

- **[GD Browser](https://gdbrowser.com/) por [GD Colon](https://github.com/GDColon)**  
Recriação da interface do Geometry Dash disponível via navegador

- **[GD Forums](https://gdforums.com/) por [Hyperbolus Team](https://github.com/hyperbolus)**  
Fórum da comunidade que pode interagir com os servidores do Geometry Dash

- **[SPWN Language](https://github.com/Spu7Nix/SPWN-language) por [Spu7nix](https://github.com/Spu7Nix)**  
Linguagem de programação criada para desenvolver níveis no Geometry Dash

- **[GD History](https://history.geometrydash.eu/) por [Cvolton](https://github.com/Cvolton)**  
Preservação da história do Geometry Dash

- **[GMDPrivateServer](https://github.com/Cvolton/GMDprivateServer) por [Cvolton](https://github.com/Cvolton)**  
Implementação de um servidor privado para os servidores do Geometry Dash

- **[Geode](https://geode-sdk.org/) por [Geode Team](https://github.com/geode-sdk)**  
Carregador de mods e framework de modding multiplataforma

### Modificações do Cliente

- **[Megahack](https://absolllute.com/store/view_mega_hack_pro) por [Absolute](https://github.com/absoIute)**  
Cliente modificado para Windows e Android que inclui centenas de mods

- **[iCreate Pro](https://geticreate.pro/) por [Camila314](https://github.com/camila314)**  
Cliente modificado para iOS que inclui mais de 30 mods

- **[Crystal](https://github.com/ninXout/Crystal-Client/tree/main) por [ninXout](https://github.com/ninXout)**  
Cliente modificado para MacOS que inclui mais de 60 mods

- **[BetterEdit](https://github.com/HJfod/BetterEdit) por [HJFod](https://github.com/HJfod/)**  
Uma extensão para o editor do GD com muitas melhorias na qualidade de vida

- **[Textureldr](https://github.com/poweredbypie/textureldr) por [PoweredByPie](https://github.com/PoweredByPie/)**  
Sistema integrado de pacotes de texturas
